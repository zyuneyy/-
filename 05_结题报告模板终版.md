# 结题报告模板终版

## 项目概述

本项目围绕复杂优化问题中的调度与路径规划开展研究，旨在构建一种由深度强化学习驱动的自适应进化算法框架，以提升传统进化算法在复杂搜索过程中的自适应调参能力。项目以作业车间调度问题和路径规划问题为主要研究对象，分别建立了标准 benchmark 实验体系，并在普通遗传算法、规则型自适应遗传算法和 DQN 驱动自适应遗传算法之间开展对比研究。研究结果表明，学习型参数控制机制在两类典型优化问题上均具有良好的应用潜力。

## 主要研究内容

1. 针对作业车间调度问题进行建模，以最小化总完工时间 `Makespan` 为目标，构建调度优化实验框架。
2. 针对路径规划问题进行建模，以旅行商问题 `TSP` 作为最小路径规划模型，以最小化总路径长度为目标，构建路径规划实验框架。
3. 设计并实现普通遗传算法 `GA`、规则型自适应遗传算法 `Adaptive GA` 以及深度强化学习驱动的自适应遗传算法 `DQN-GA`。
4. 构建基于 `PyTorch` 的 DQN 参数控制器，根据种群状态动态调整交叉率和变异率。
5. 基于标准 benchmark 数据集完成实验验证，并通过图表与可视化方式展示算法效果。

## 实施过程

项目首先对调度与路径规划问题进行了形式化建模，并完成 benchmark 数据接入。调度部分选取 `FT06、FT10、LA01-LA10` 等经典作业车间调度实例，路径规划部分选取 `burma14` 与 `berlin52` 等 TSPLIB 风格实例。随后围绕上述问题分别完成了 `GA` 和 `Adaptive GA` 的实现，形成稳定可复现的算法基线。在此基础上，进一步使用 `PyTorch` 构建 DQN 控制器，将强化学习嵌入进化算法的迭代过程，使其能够根据当前最优值、平均值、种群多样性和停滞代数等状态信息动态选择参数组合。最后，通过多种随机种子重复实验，对三类算法进行了系统对比，并输出了收敛曲线、甘特图、路径图和动画等展示结果。

## 主要成果

1. 完成了调度主线的算法实现与实验验证，建立了 `GA / Adaptive GA / DQN-GA` 三类方法的完整比较体系。
2. 在调度 baseline 实验中，`Adaptive GA` 在多个标准实例上稳定优于普通 `GA`，例如在 `FT10` 上，`Adaptive GA` 平均结果为 `1067.2`，优于 `GA` 的 `1089.0`。
3. 在调度专项实验 `LA02` 上，`DQN-GA` 的平均结果为 `714.6`，优于 `Adaptive GA` 的 `743.0` 与 `GA` 的 `724.0`，验证了学习型调参机制优于人工规则调参。
4. 完成了路径规划支线的算法实现与实验验证，建立了 `GA / Adaptive GA / DQN-GA` 的路径规划实验体系。
5. 在路径规划实例 `berlin52` 上，经强化训练后，`DQN-GA` 的平均结果达到 `7910.4`，优于 `Adaptive GA` 的 `8025.2` 和 `GA` 的 `8090.8`，表明学习型参数控制机制同样适用于路径规划问题。
6. 完成了多类可视化成果，包括调度甘特图、收敛曲线、对比柱状图、调度动画以及路径规划路径图，为项目答辩与结果展示提供了直观支撑。

## 创新点

1. 将深度强化学习引入进化算法参数控制过程，形成了“强化学习驱动进化算法自适应调参”的研究框架。
2. 在调度与路径规划两类典型优化问题上完成了双线验证，说明该机制具有一定的跨问题适用潜力。
3. 在调度 `LA02` 和路径规划 `berlin52` 上均实现了 `DQN-GA` 优于 `Adaptive GA` 的实验结果，验证了学习型参数控制超过人工规则的可能性。

## 存在问题

1. 当前路径规划部分采用的是最小模型 `TSP`，尚未扩展到更复杂的 `CVRP` 或 `VRPTW`。
2. 多目标优化方法尚未作为本次结题的主实验内容正式展开。
3. 项目已完成可视化展示原型，但尚未开发完整交互式应用系统。
4. DQN 模型的泛化能力和跨实例稳定性仍有进一步优化空间。

## 经费使用

项目经费主要用于资料检索、文献调研、算法实验所需的软件与计算资源支持、答辩与成果整理等相关开支。项目总体经费使用遵循节约、合理和服务研究任务的原则，能够满足项目实施需求。

## 结论

本项目围绕“深度强化学习驱动的自适应进化算法”开展了较系统的研究，完成了调度与路径规划两条研究主线的算法设计、程序实现、实验验证与结果展示。实验表明，`DQN-GA` 不仅在调度 `LA02` 上优于规则型 `Adaptive GA`，也在路径规划 `berlin52` 上取得了更优结果，验证了学习型参数控制机制在典型组合优化问题中的有效性。总体来看，项目已实现预期核心目标，并为后续开展多目标优化、更复杂路径规划模型研究以及应用原型开发奠定了基础。
